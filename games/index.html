ReadMe

For this project, since the suggested topic was games I thought I would do something related to them but I couldn't think of any games that I could collect data from and then predict stuff so instead I did some research on some classic board games, as well as some more current ones. Originally when I came up with this idea I had planned to compare what the creators intended vs. reality. For example, the creator usually has an idea of how long the game will take to play and typically this information is included in the instructions. However, this intended time does not always prove to be true, especially in situations like Monopoly. The intented time for this game is roughly 2 hours but I have definitly played for less and more time than that. Usually someone gets angry and flips the board or something wayyy before 2 hours or in the rare case that everyone is mature the game can literally last forever. I have never actually witnessed a game finish the way the creater intended, in my experience someone is always bitter about losing properties to the roll of the dice. ANYWAY, to do these comparisons would require playing all of these games and I simply did not have time to do that. Therefore, I thought it would be interesting to visualization the demographics related to board/card games.

Understanding the visualization. When designing the visualization I thought it would be fun to make it resemble a game board. Colored squares are a popular design aspect of board games--something else I learned doing this project--they are often used to represent the spaces. I created visualizations based on 4 different filters. The first is the minimum number of players required to play each game. The second is the maximum number of players allowed for each game. The third is based on the creaters intended play time. The last filter relates to the age demographic the game is target for. While doing this research I learned that the most games are created for 2-4 players, intended to last no more than 30 minutes and are targeted for ages 6 and up.

For each game I included the information I found. If you hover over the square on the right the sticky note will populate with the information of the game that square represents. Another feature that I did not end up including is filtering the games by different categories or tags but I still included this information on the right in the sticky note.
